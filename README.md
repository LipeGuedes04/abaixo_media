# 📊 Elementos Abaixo da Média em um Vetor (Java)

## 📌 Descrição

Este projeto consiste em uma aplicação em Java que calcula a média dos valores armazenados em um vetor e exibe os elementos que estão abaixo dessa média.

O programa solicita ao usuário a quantidade de elementos, realiza a leitura dos valores e, após calcular a média, identifica e apresenta apenas os números inferiores ao valor médio.

---

## 🚀 Funcionalidades

* Definição do tamanho do vetor
* Leitura de números reais
* Cálculo da média dos elementos
* Identificação dos valores abaixo da média
* Exibição dos resultados

---

## 🧱 Estrutura do Programa

O programa é composto por uma única classe:

* `abaixo_da_media`: responsável pela leitura dos dados, cálculo da média e filtragem dos valores

---

## 🖥️ Tecnologias Utilizadas

* Java
* Vetores (arrays)
* Estrutura de repetição `for`
* Estrutura condicional `if`
* Classe `Scanner` para entrada de dados
